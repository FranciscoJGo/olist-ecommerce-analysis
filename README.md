## 🇲🇽 Español

### Descripción
Análisis exploratorio de datos de ventas del marketplace brasileño Olist (2017-2018), 
con más de 96,000 órdenes entregadas y R$ 19.7 millones en revenue total.

### Hallazgos principales
- 📈 **Crecimiento constante** de revenue mes a mes durante 2017-2018
- 🛍️ **Top categorías:** bed_bath_table y health_beauty lideran en volumen de ventas
- 📅 **Patrón de compra:** Los clientes compran principalmente de lunes a jueves
- 💳 **Ticket más alto:** watches_gifts y computers_accessories (mayor valor por orden)
- 🎯 **Ticket promedio:** R$ 204.98 por orden

### Tecnologías
- Python 3.13, pandas, matplotlib, seaborn, JupyterLab

## 🌎 English

### Description
Exploratory data analysis on Olist Brazilian e-commerce dataset (2017-2018),
covering 96,000+ delivered orders and R$ 19.7M in total revenue.

### Key Findings
- 📈 **Consistent revenue growth** month over month throughout 2017-2018
- 🛍️ **Top categories:** bed_bath_table and health_beauty lead in sales volume
- 📅 **Purchase pattern:** Customers buy primarily Monday through Thursday
- 💳 **Highest ticket:** watches_gifts and computers_accessories
- 🎯 **Average order value:** R$ 204.98

### Tech Stack
- Python 3.13, pandas, matplotlib, seaborn, JupyterLab
